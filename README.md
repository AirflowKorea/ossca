## 2026 Apache Airflow 멘티 PR 현황

- 표의 **작성 PR / 리뷰 PR** 숫자를 클릭하면 해당 멘티의 GitHub 검색 결과로 이동합니다.
- 각 팀의 **상세 PR 목록**을 펼치면 PR 제목·상태·생성일이 함께 표시됩니다. 상태: 🟢 open / 🟣 merged / 🔴 closed.
